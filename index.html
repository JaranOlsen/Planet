Movement control:

O Switch between Orbit controls and Fly controls
, In Fly controls: Switch between point-to-look and drag-to-look

Slide control:

← Previous slide
→ Next slide

Light control:

1-5 Spotlight intensity
6-0 Ambient light intensity

Boids control:

G Boids stats
H Boids parameters control
J FPS counter

Content display:

C Show/hide planet content
I Show/hide planet interior
S Show/hide spiral dynamics

Developer tools:

Z Clear selection of nodes
A Add to selection of nodes
F Toggle fast mode
Q Print nodedata
T Create new node
X Remove selected node(s)
R Create connection(s) to first selected node
V Create arrowed connection(s) to first selected node
B Create dashed connection(s) to first selected node
N Create tunneled connection(s) to first selected node
W Redraw connections
E Change color (next) of selected node(s)
D Change color (previous) of selected node(s)
+ Increase size of selected node(s)
- Decrease size of selected node(s)
Arrows Move selected node(s)
Click nodes to see content. Wireframe nodes doesn't have content. Press K for hotkeys.
